Getting Started With Python

Faculty Technology Development, 2018

Overview

We need to install an interactive text editor, the software to actually run Python code, and a package to run the code from within the text editor. The steps that follow are for a Windows machine, although they are similar for a Mac. Let me know if you run into problems following these steps!

Installing the Atom text editor

Atom is a free text editor with advanced capabilities. (I'm actually writing this handout in Atom, see the screenshot below.)

  1. Go to here and click Download. It should automatically detect what kind of computer you're using.
  2. Save the file to your Desktop (or wherever ....)
  3. Double-click on the file to open the installer. A cute little animation will open that says "Atom is being installed. It will launch once it's done." It should open when it's done.
  4. Click "Yes, Always" if it asks you whether it should be the default atom URL handler.
  5. You can play around if you like, but now we need to get Python installed.

Installing Python

Python is a programming language, so we need a way for your computer to run it. We're going to use Anaconda, which makes installing Python simple.

  1. Go to here and under "Python 3.7 version" click Download.
  2. Save the file to your Desktop. It's a large file (631MB), so it may take a few minutes to download. (You don't need to give your email address to get a "cheat sheet" as will popup on the webpage after you download.)
  3. Double-click on the file. Hit Next, I Agree, Next, Next for the destination folder, select both Advanced Options, and click Install.
  4. A green bar will popup showing the installation progress. You can probably pause and go work on something else, as this will probably take more than a few minutes.
  5. When it says Completed, hit Next, Skip, unselect both "learn more" boxes and click Finish.

Running Python from Atom

Now we're going to get Atom to play nicely with Python!

  1. Go back to Atom, which should still be open on your computer.
  2. In Atom, click on the tab labeled "Untitled." Type this in 
    3. # My first Python program
  print('Hello, Truman!')
  3. Click Control-S to save the file. Just save it to your Desktop for now, call it "test.py".
  4. Your code should now change colors as Atom recognizes it as a Python file. It will be pretty.
  5. There should be a pane open that says "Get to know Atom." In that pane, click on "Install a Package" then "Open Installer".
  6. In the Search Packages box, type in "Hydrogen."
  7. Under Hydrogen 2.6.0, click Install. It will take a few minutes.
  8. Now, click in the Windows search box at the bottom of the screen, and type in Command Prompt. Open the command prompt, and type the following in:
    9. python -m pip install ipykernel
  9. After that runs, type the following in the command prompt:
    10. python -m ipykernel install --user
  10. Close the command prompt.
  11. Close Atom, then re-open it.
  12. Once it's open, click back on your file "test.py", which will show up in a tab at the top of the editor.
  13. Click on what you typed in, and hit Control-Enter.
  14. The code should run and print "Hello, Truman!" off to the side.
  15. On a new line, type "2+2" and hit Control-Enter. It should display "4" to the side. Now you can do math.

Congrats, now we're ready to code!
